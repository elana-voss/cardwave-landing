Cardwave

本地优先的角色卡编辑器与聊天客户端。接入你已在使用的任意 AI 服务商。除非你主动发送，否则任何数据都不会离开你的电脑。

隐私

Cardwave 将角色卡、聊天、密钥和设置保存在你的磁盘上。无需账号，不同步，应用内无数据统计。唯一的对外请求由你触发：聊天消息发送给你填入 API 密钥的服务商。传输路径上没有任何 Cardwave 服务器。

模型

Cardwave 接收 API 密钥并直接与服务商通信。每个聊天可切换服务商。Cardwave 没有订阅收费。

  • platform.openai.com GPT- OpenAI
  • console.anthropic.com Claude - Anthropic
  • aistudio.google.com Gemini- Google
  • console.x.ai Grok - xAI
  • openrouter.ai OpenRouter- 多种模型
  • nano-gpt.com NanoGPT - 多种模型

角色卡

编辑器完整覆盖 Character Cards V3 规范。把 PNG 拖进角色卡文件夹，即可在编辑器或聊天中打开。

去哪里找角色

Cardwave 可以打开来自任何地方的角色卡。查看托管角色卡的网站列表，下载后放入你的角色卡文件夹即可。

费用估算

聊天费用会随对话变长而增加，因为每一轮都会重新发送历史记录。聊天轮次预算计算器可按任意模型的每 token 价格，估算一笔预算能聊多少轮。

CCV4 卡片格式

CCV4 是一个草案阶段的角色卡格式，由 TOML 编写的小型可寻址文本块组成，存储在 PNG 数据块中。阅读 CCV4 概览，包含完整规范、提示词组装算法、参考示例和在线演练场。
