Cardwave

A local-first character card editor and chat client. Bring any AI provider you already use. Nothing leaves your computer unless you send it there.

Privacy

Cardwave stores cards, chats, keys, and settings on your disk. No account, no sync, no analytics in the app. The only outbound calls are the ones you trigger: a chat message going to the provider whose API key you entered. There is no Cardwave server in the path.

Models

Cardwave takes an API key and talks to the provider directly. Swap providers per chat. There is no Cardwave subscription.

  • platform.openai.com GPT- OpenAI
  • console.anthropic.com Claude - Anthropic
  • aistudio.google.com Gemini- Google
  • console.x.ai Grok - xAI

  • nano-gpt.com NanoGPT - diverse models

Cards

The editor covers the full Character Cards V3 spec. Drop a PNG into the cards folder and it opens in either the editor or the chat.

Where to find characters

Cardwave opens cards from anywhere. See the list of sites that host character cards you can download and drop into your cards folder.

Estimating costs

Chat costs grow as a conversation lengthens, because each turn resends the history. The chat turn budget calculator estimates how many turns a budget buys for any model's per-token price.

Scope

Cardwave is free and built by one person. Released as-is. Bug reports and feature requests welcome via GitHub Issues.

Beta. Cardwave is in active development. Keep backups of your character cards.

CCV4 card format

CCV4 is a draft character card format built from small addressable text blocks in TOML, stored in PNG chunks. Read the CCV4 overview, with the full specification, the prompt-assembly algorithm, a reference example, and a live playground.
