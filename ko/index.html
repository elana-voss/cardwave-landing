프라이버시

Cardwave는 카드, 채팅, 키, 설정을 내 디스크에 저장합니다. 계정 없음, 동기화 없음, 앱 내 분석 없음. 외부로 나가는 요청은 내가 직접 트리거한 것뿐입니다: 입력한 API 키의 제공사로 전송되는 채팅 메시지. 그 경로에 Cardwave 서버는 없습니다.

모델

Cardwave는 API 키를 받아 제공사와 직접 통신합니다. 채팅마다 제공사를 바꿀 수 있습니다. Cardwave 구독은 없습니다.

카드

에디터는 Character Cards V3 사양을 완전히 지원합니다. PNG를 카드 폴더에 넣으면 에디터나 채팅에서 바로 열립니다.

캐릭터 찾는 곳

Cardwave는 어디서 가져온 카드든 열 수 있습니다. 다운로드해서 카드 폴더에 넣을 수 있는 캐릭터 카드 호스팅 사이트 목록을 확인하세요.

비용 추정

채팅 비용은 대화가 길어질수록 커집니다. 매 턴마다 히스토리를 다시 전송하기 때문입니다. 채팅 턴 예산 계산기는 모델별 토큰 단가로 예산이 몇 턴을 감당하는지 추정합니다.

프로젝트 범위

Cardwave는 무료이며 한 사람이 만듭니다. 있는 그대로 배포됩니다. 버그 신고와 기능 요청은 GitHub Issues로 환영합니다.

베타. Cardwave는 활발히 개발 중입니다. 캐릭터 카드는 백업해 두세요.

CCV4 카드 포맷

CCV4는 TOML로 된 작고 주소 지정 가능한 텍스트 블록을 PNG 청크에 저장하는 초안 단계의 캐릭터 카드 포맷입니다. 전체 사양, 프롬프트 조립 알고리즘, 참조 예제, 라이브 플레이그라운드가 포함된 CCV4 개요를 읽어 보세요.
